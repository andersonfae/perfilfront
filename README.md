## Como Iniciar o Projeto

Para executar este projeto em sua máquina local, siga estas etapas:

1. **Clone o Repositório:**

   ```bash
   git clone https://github.com/seu-usuario/seu-projeto.git
   cd perfil

   ```

2. **Instale as dependencias:**
   npm install

3. **Rode a aplicação:**
   npx next
